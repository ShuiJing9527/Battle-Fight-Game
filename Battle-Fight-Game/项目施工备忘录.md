## 五、摄像机逻辑重写

旧摄像机跟随逻辑已弃用并删除：

- `CenteredCameraFollow.cs`
- `CenteredCameraFollow.cs.meta`

新增：

`Assets/Script/摄像机/PlayerCameraRig.cs`

当前摄像机逻辑：

- 只认 `playerSlot`
- 不自动搜索目标
- 每帧强制跟随 Player
- 每帧看向 Player 中心点
- 角色保持居中
- 可以看到天空

`草原/Main Camera` 当前挂载：

- `PlayerCameraRig`

当前参数：

- `Player Slot = Player`
- `Distance = 18`
- `Height = 4`
- `Yaw = 0`
- `Center Offset = (0, 1, 0)`
- `Lock Every Frame = true`

核心逻辑：

```csharp
targetCenter = Player.position + CenterOffset;
camera.position = targetCenter + new Vector3(0, Height, -Distance);
camera.LookAt(targetCenter);
```

## 七、后续注意事项

如果角色仍不可见，优先检查：

1. 当前运行的是否是 `草原` 场景
2. `Main Camera` 上的 `PlayerCameraRig.playerSlot` 是否绑定了 `Player`
3. `Player` 是否处于激活状态
4. `Main Camera` 是否是当前启用的主摄像机
5. 是否有旧存档把场景加载回了 `GameScene`
